Default checkpoint is Felladrin/Smol-Llama-101M-Chat-v1 (Llama-arch, ~101M params). Click "load model + tokenizer" first; then enter a prompt and "generate". The benchmark button still uses a tiny synthetic model to compare backends without a real-weight load.

Training demo (synthetic transformer)

Trains a tiny in-process transformer (1 layer, 8 hidden, 8 vocab) with cross-entropy loss and Adam to memorize the sequence [1,2,3] → [2,3,4]. Loss should drop from ~2.0 toward zero. Initial / final L2 norm reported via Jax.Pytree's heterogeneous traversal.

   

Karpathy-style microGPT, in-browser. Trains a 1-layer transformer on the corpus below (one token per character), then samples continuations from a single-character prompt. See examples/microgpt/ for the matching CLI demo and the side-by-side Faithful (from-scratch autograd) port.

       

With these defaults (100 steps, lr=5e-3, T=0.8) the model memorizes the first ~32 chars of the corpus; samples often play back leading names like "ada", "emma". Bump steps to 500+ and add corpus variety for genuine generation.
